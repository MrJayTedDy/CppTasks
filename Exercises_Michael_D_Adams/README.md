# General overview
This folder contains the implementation for exercises created by Michael D. Adams. listed in the book:
Exercises for Programming in C++ (Version 2021-04-01)

# Stucture
Each folder named as the chapter in the book.

Each folder contains different source files:
 - Each file is named as a number of the task in the book
 - Each file has a description of the task at the beginning of the file as a comment
 - If task requires creation more than 1 source files then those files will be placed inside the folder
   which will be named as the number of the task
